About MatchMySkills

MatchMySkills is a simple yet powerful web platform designed especially for students, freshers, and professionals who want to find job opportunities that truly fit their skills. Unlike many traditional job portals that rely mainly on job titles or keywords, MatchMySkills digs deeper by analyzing the skills extracted directly from your resume, providing personalized and relevant job matches.

This project uses a combination of advanced tools and modern technologies to deliver this experience:

  • The backend is built with Python and Flask, a lightweight web framework that handles user authentication, resume upload, and job matching logic seamlessly.
  • For resume processing, it uses PyMuPDF to extract text from uploaded PDF resumes safely and efficiently.
  • It leverages spaCy, a powerful Natural Language Processing (NLP) library, to analyze the resume text and extract important skill phrases and entities.
  • To find the best job matches, it employs Sentence Transformers models (like all-MiniLM-L6-v2) to convert resumes and job descriptions into mathematical vectors and then compares these vectors using semantic similarity — meaning matches are based on the meaning of skills, not just exact keywords.
  • Job listings are fetched in real-time from the Adzuna Job Search API, focusing on roles and locations relevant to the user.
  • The frontend uses clean HTML and Jinja2 templates for forms and results display, styled with custom CSS for a user-friendly interface inspired by GitHub’s dark theme.
  • The project is organized with a Python virtual environment (venv) to manage dependencies like Flask, spaCy, PyMuPDF, requests, and sentence-transformers, ensuring a smooth development and deployment process.
